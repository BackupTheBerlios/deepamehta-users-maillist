Liebe DeepaMehta-Freunde,

Ab sofort steht der 3. Release Candidate von DeepaMehta 2.0b8 zur 
Verfügung: www.deepamehta.de/download/deepamehta-2.0b8-rc3.zip

Neu im 3. Release Candidate:

=> Terminal-lose Automatik-Installation auch für Mac OS X (vorher nur 
für Windows): Nach dem Auspacken genügt ein Doppelklick zum 
Programmstart
=> Gestärkter Social-Software Aspekt: die User können ihr Profil selbst 
ändern (der User-Topic ist editierbar)
=> Erweitertes README (Mac OS X Quickstart, DeepaMehta updaten)
=> 1. Version des Englischen README!
=> Bug Fix für Windows: Installation funktioniert auch dann, wenn der 
Install-Pfad Leerzeichen enthält
=> Bug Fix: Keine Merkwürdigkeiten mehr im Zusammenhang mit Workspace 
Join/Leave.

Die wesentlichen Neuerungen von DeepaMehta 2.0b8 gegenüber DeepaMehta 
2.0b7 sind im Anhang beschrieben.

Was derzeit noch fehlt und in der finalen 2.0b8 Release vorhanden sein 
wird:
- Komplettiertes README (DB-Reset, DeepaMehta deinstallieren)
- Online-Hilfe (Topictyp-Beschreibungen)
- Aktualisierter Beispiel-Content (DeepaMehta-Architektur-Bild)
- Umarrangierte Icons ("Hide", "Close", "Remove")
- Bug Fixes (Import/Export, Kalendaranwendung, Kompetenzsternanwendung)

Probiert eine lokale DeepaMehta Installation! Funktioniert es mit 
Windows, Linux und Mac OS X? Ist das README verständlich? Bleiben 
Fragen offen? Probiert auch die mitgelieferten Beispiel-Anwendungen 
(Kompetenzstern, Film-DB-Anbindung, ...) und Beispiel-Webfrontends 
unter Tomcat (Corporate Memory Browser, Topicmap-Viewer, Forum ...)! 
Klappt die Installation? Funktionieren die Anwendungen?

Euer Feedback kann noch in die finale Release einfließen. Die finale 
DeepaMehta 2.0b8 Release soll in diesem Monat erscheinen.

Auf www.deepapmehta.de ist noch das alte DeepaMehta 2.0b7 installiert. 
Noch in dieser Woche soll der 3. Release Candidate auch für die 
Online-Nutzer zur Verfügung stehen.

*** DANKE an Harald Müller für die Englisch-Übersetzung des README! ***
*** Danke auch an Matthias Melcher für den Windows-Test und Feedback! 
***

Grüße
Jörg

Neu im 2. Release Candidate (12.3.2008):

=> Installationsroutine funktioniert auch für Windows (NT, 2000, XP, 
Vista). Nach dem Auspacken genügt ein Doppelklick zum Starten.
=> Weitere Vereinfachung der Installation (für alle Plattformen). Eine 
Automatik-Installation erspart manuelles Konfigurieren und 
Installieren.
=> Erweitertes README (Administration, Client-Server, Web-Frontends)
=> Neue Optik durch neue (Auge-)Icons
=> Paketgröße von 26 MB auf 12 MB reduziert

Neu im 1. Release Candidate (26.2.2008) / Wesentliche Neuerungen 
gegenüber der alten 2.0b7 Version:

=> Vereinfachte Installation. Insbesondere muß kein MySQL und kein 
"ant" mehr installiert werden.
=> Neue Standardanwendung: Gruppenkalender. Wochenansicht, 
Monatsansicht, Verabredungen, Events, Teilnehmer, Orte, ...
=> Erweiterte Kollaborationsfunktionen. Gemeinsames Arbeiten an 
Topic-Inhalten (neue Rollen: "Editor", "Publisher")
=> Für Entwickler: mehr Flexibilität im HTML-Generator 
(Topic-Info-Generator)
=> Für Entwickler: komfortables Build- und Konfigurations-Management
