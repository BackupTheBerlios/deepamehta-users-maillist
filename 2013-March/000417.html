Dear DeepaMehta users,

On Mar 11, 2013 DeepaMehta 4.1 has been released. This is the most important DM4 release so far as it comes with a data sustainability guarantee.

For the first time, data you enter in DM4 is guaranteed to be transferred to all future DM4 releases automatically. So, the DM4 experimental stage is over, and you can start productive DM4-based projects now.

For a complete list of changes in DM 4.1 and further details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

Updating from previous DeepaMehta versions is *not* supported. Sorry.
But now with DM 4.1 you will not read this sentence again :-)

Donwnload:
http://demo.deepamehta.de/download/deepamehta-4.1.zip			(Standard Distribution)
http://demo.deepamehta.de/download/deepamehta-4.1-karaf.tar.gz	(Karaf Distribution)

For installation instructions see the README (Standard Distribution):
https://github.com/jri/deepamehta#readme

For setting up the Karaf Distribution refer to
https://trac.deepamehta.de/wiki/KarafDistribution

You can try out DeepaMehta online with our live demo server.
No download required. No registration required.
http://demo.deepamehta.de/

For reporting issues please register at our trac-site:
https://trac.deepamehta.de

Thanks to Jürgen Neumann, Danny Gräf, Malte Reißig, Silke Meyer, Thilo Schönnemann, Jörn Weißenborn and Rolf Kutz!

BTW: we're celebrating the advent of DM 4.1 at the next DeepaMehta Community Lounge on Mar 28, 2013 in Berlin. See http://www.deepamehta.de/de/content/4th-deepamehta-community-lounge

Cheers,
Jörg
