Dear DeepaMehta users,

The release of DM 4.1 is imminent. This is the most important DM4 release so far as it is the first one that comes with a sustainability guarantee for your data.

Please help us with thoroughly testing the upcoming release.

You can download the current development version ("nightly build") here:
http://demo.deepamehta.de/files/deepamehta-standard-4.1-SNAPSHOT-2013-03-08-nightly.zip
Respectively download the latest nightly build from here:
http://demo.deepamehta.de/

Alternatively you can try out the nightly build online (without downloading anything) here:
http://demo.deepamehta.de:8083/de.deepamehta.webclient
Login as "admin" and leave the Password empty.

We think that the nightly build is very close to the state that will be released as DM 4.1

To guide your tests the Release Notes will published in the next days. The Release Notes tell you about the new features and changes. In the meantime you could test weather the known features are still working. You can track the recent changes on the Trac timeline:
https://trac.deepamehta.de/timeline
(This is extensive information. To filter just the important ones you could uncheck all "Changesets" and press "Update". This brings up just the tickets.)

To report an issue please open a ticket on the Trac site:
https://trac.deepamehta.de/
(You need to register. Press "Login" first.)

Thank you.

Cheers,
Jörg
