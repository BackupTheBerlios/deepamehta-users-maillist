Dear DeepaMehta Community,

Danny has published a new DeepaMehta4 Plugin that enables CSV import for
simple TopicTypes = any composite of an arbitrary number of level one
aggregations and/or compositions with ordinary cardinality (in other
words: a simple object with n properties).

Read more here: https://github.com/dgf/dm4-csv

Download:
http://download.deepamehta.de/nightly/deepamehta-csv-0.0.1-SNAPSHOT.jar

Please give it a try and report back!
