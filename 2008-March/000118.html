Liebe DeepaMehta-Freunde,

Ab sofort steht der 2. Release Candidate der bevorstehenden DeepaMehta 
2.0b8 Release zur Verfügung: 
www.deepamehta.de/download/deepamehta-2.0b8-rc2.zip

Was ist neu seit dem 1. Release Candidate (26.2.2008)?
=> Installationsroutine funktioniert auch für Windows (NT, 2000, XP, 
Vista)
	Nach dem Auspacken genügt ein Doppelklick zum Starten
=> Weitere Vereinfachung der Installation (für alle Plattformen)
	Eine Automatik-Installation erspart manuelles Konfigurieren und 
Installieren
=> Erweitertes README (Administration, Client-Server, Web-Frontends)
=> Neue Optik durch neue (Auge-)Icons
=> Paketgröße von 26 MB auf 12 MB reduziert

Was ist neu seit DeepaMehta 2.0b7 (25.6.2006)?
=> Vereinfachte Installation
	Insbesondere muß kein MySQL und kein "ant" mehr installiert werden.
=> Neue Standardanwendung: Gruppenkalender
	Wochenansicht, Monatsansicht, Verabredungen, Events, Teilnehmer, Orte, 
...
=> Erweiterte Kollaborationsfunktionen
	Gemeinsames Arbeiten an Topic-Inhalten (neue Rollen: "Editor", 
"Publisher")
=> Für Entwickler: mehr Flexibilität im HTML-Generator 
(Topic-Info-Generator)
=> Für Entwickler: komfortables Build- und Konfigurations-Management

Was derzeit noch fehlt und in der finalen Release vorhanden sein wird:
- Englisches README
- Möglicherweise: Online-Hilfe (Topictyp-Beschreibungen)

Probiert eine lokale Installation! Ein deutsches README liegt bei. Ist 
das README verständlich? Bleiben Fragen offen?

Probiert auch die mitgelieferten Beispiel-Anwendungen (Kompetenzstern, 
Film-DB-Anbindung, ...) und Beispiel-Webfrontends unter Tomcat 
(Corporate Memory Browser, Topicmap-Viewer, Forum ...)! Klappt die 
Installation? Funktionieren die Anwendungen?

Euer Feedback kann noch in die finale Release einfließen.
Die finale DeepaMehta 2.0b8 Release ist für Montag, 17. März geplant.

Auf www.deepapmehta.de ist noch die alte Version installiert. In der 
kommenden Woche wird dann die finale 2.0b8 Release auch für die 
Online-Nutzer zur Verfügung stehen.
