Liebe DeepaMehta-Freunde,

an der neuen DeepaMehta Release 2.0b8 wird unter Hochdruck gearbeitet. 
Ab sofort steht ein erster Release Candidate zur Verfügung:
www.deepamehta.de/download/deepamehta-2.0b8-rc1.zip

Features der neuen Release:
=> Vereinfachte Installation
	Insbesondere muß kein MySQL und kein "ant" mehr installiert werden.
=> Neue Standardanwendung: Gruppenkalender
	Wochenansicht, Monatsansicht, Verabredungen, Events, Teilnehmer, Orte, 
...
=> Erweiterte Kollaborationsfunktionen
	Gemeinsames Arbeiten an Topic-Inhalten (neue Rollen: "Editor", 
"Publisher")
=> Für Entwickler: mehr Flexibilität im HTML-Generator 
(Topic-Info-Generator)

Probiert eine lokale Installation! Ein deutsches README liegt bei.

Probiert auch die mitgelieferten Beispiel-Anwendungen (Kompetenzstern, 
Film-DB-Anbindung, ...) und Beispiel-Webanwendungen unter Tomcat 
(Corporate Memory Browser, Topicmap-Viewer, Forum ...)! Euer Feedback 
wird dann in den nächsten Release Kandidaten einfließen. Die finale 
2.0b8 Release ist für Montag, 3. März geplant.

Was derzeit noch fehlt und in der finalen Release vorhanden sein wird:
- Installationsroutine für Windows (derzeit nur Linux und Mac OS X)
- Erweitertes README (Administration, Client-Server, Web-Frontends)
- Englisches README
- Möglicherweise: Online-Hilfe (Topictyp-Beschreibungen)

Auf www.deepapmehta.de ist noch die alte Version installiert. In der 
kommenden Woche wird dann die finale 2.0b8 Release auch für die 
Online-Nutzer zur Verfügung stehen.

Grüße
Jörg
