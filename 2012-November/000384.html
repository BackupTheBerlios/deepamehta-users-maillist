Here I like you to inform about the upcoming DeepaMehta 4.0.13 release.

Main features will include:

1) Performance.
The current 4.0.12 version shows bad performance due to introduction of Access Control. The upcoming 4.0.13 uses a different approach for storing the Access Control information and will return to the performance known from before.

2) For the first time DM will come in two flavors:
	- DeepaMehta Standard Distribution
	- DeepaMehta Enterprise Distribution
The Standard Distribution is what you already know. Its focus is small download size and easy setup for single users. The Enterprise Distribution (name remains to be discussed) focuses on running and monitoring a DM client-server installation for collaborative work. The Enterprise Distribution is based on Apache Karaf (http://karaf.apache.org/)

3) GUI improvements:
	- Canvas: Moving clusters of connected nodes
	- Detail panel: the association listing is grouped by association type
	- Create associations via Shift-Drag.

DeepaMehta 4.0.13 is planned to be released at the end of November.
