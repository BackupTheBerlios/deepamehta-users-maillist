Dear everybody!

The latest DeepaMehta release 4.0.9 is now packaged for Debian/Ubuntu.
The package contains the geomaps and facets plugins.

As before, you can install it from the following repository:

deb http://ppa.launchpad.net/sil-ke/deepamehta4/ubuntu
YOUR_UBUNTU_VERSION_HERE main

Ubuntu users can also use the following command:
add-apt-repository ppa:sil-ke/deepamehta4 to add it to your sources.list.

If you'd like to have a closer look to what's inside please refer to
https://github.com/silke/A-Debian-package-for-DeepaMehta where I
uploaded all files needed to build the package. Feedback welcome!

Best,
Silke
