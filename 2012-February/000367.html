On Feb 3, 2012 DeepaMehta 4.0.9 has been released.

This is a pure bug fix release that fixes the new CKEditor (introduced in yesterday's 4.0.8 release).
CKEditor now works in non-english environments.

For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

You *can* update from DeepaMehta 4.0.7 or later and reuse your content.
See the README for update instructions.

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.9.zip

README with installation and update instructions:
https://github.com/jri/deepamehta#readme

List of available plugins:
https://trac.deepamehta.de/wiki/Plugins

Build from source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de

Thanks to Matthias Melcher for bug reporting!

Cheers,
Jörg
