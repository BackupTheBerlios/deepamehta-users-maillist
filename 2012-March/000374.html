On Mar 24, 2012 DeepaMehta 4.0.10 has been released.
This minor update just contains 2 bug fixes.

Bug fixes:
- The Back button works cross-topicmap again (#231).
- Building from source with a pristine maven repo works again (#234).

For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

You *can* update from DeepaMehta 4.0.7 or later and reuse your content.
See the README for update instructions.

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.10.zip

README with installation and update instructions:
https://github.com/jri/deepamehta#readme

List of available plugins:
https://trac.deepamehta.de/wiki/Plugins

Build from source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de

Due to other work like building the new www.deepamehta.de website the DeepaMehta 4.0.10 release took a comparatively long time. Now we're back on the development track.

Cheers,
Jörg
