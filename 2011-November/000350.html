On Nov 27, 2011 DeepaMehta 4.0.6 has been released.

This release has mostly improvements for plugin developers. It is compatible with 2 new external plugins (see below). There are some GUI improvements though.

* New feature for plugin developers:
    * Map type extension architecture: plugins can provide additional map types, e.g. a geo map or a time map.
      Each map type can have a distinguished data model, rendering facility, and interaction style.

* Improvements/Fixes:
    * Topicmaps remember their translation.
    * Menus opened within dialog boxes are not constrained to the dialog box dimension.

For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.6.zip

You must perform a clean install. You can't use content from a previous DeepaMehta installation.

Installation instructions:
https://github.com/jri/deepamehta#readme

Build from source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de

DeepaMehta 4.0.6 is compatible with 2 new plugins (optional install):

    * DM4 Geomaps: displays topics on a geographical map.
      https://github.com/jri/dm4-geomaps

    * DM4 Facets: introduces multi-typing to the DM data model.
      https://github.com/jri/dm4-facets

There is a new Wiki page aimed to collect a list of available plugins:
https://trac.deepamehta.de/wiki/Plugins

Thanks to Christiane Müller, Danny Gräf, Ingo Rau, Jürgen Neumann, Malte Reißig, Silke Meyer, and Rolf Kutz for contributing!

Cheers,
Jörg
