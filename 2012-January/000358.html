On Jan 19, 2012 DeepaMehta 4.0.7 has been released.

The main feature of this release is the compatibility with the "DM4 Kiezatlas 2.0" plugin and the attendant plugin development framework extensions. Furthermore there are some general GUI improvements and bug fixes.

Compatibility with 1 new plugin (optional install):
	* DM4 Kiezatlas 2.0: a geographical content management system.
Compatibility with 2 updated plugins (optional install):
	* DM4 Geomaps 0.2: displays geo-related topics on a geographical map.
	* DM4 Facets 0.2: introduces multi-typing to the DeepaMehta data model.
GUI improvements:
	* Auto-positioned topics appear near selection. This results in less mouse drags and less canvas clutter.
	* New icon for "search result" that replaces the bucket.

More changes and details (also for developers) are described in the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

For plugin installation instructions see the links in the release notes.

You can't reuse content created in DeepaMeht 4.0.6 or any previous DeepaMehta installation.

https://github.com/downloads/jri/deepamehta/deepamehta-4.0.7.zip
