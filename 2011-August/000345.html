On August 22, 2011 DeepaMehta 4.0.3 has been released.

After a long series of backend changes the focus of this release is to bring the frontend on a par. This release delivers 2 new featues and a lot of GUI improvements and fixes. For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.3.zip

Installation instructions in the README:
https://github.com/jri/deepamehta#readme

Build from Source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de


Thanks to Jürgen Neumann, Ingo Rau, Danny Gräf, and Malte Reißig for contributing to this release!

Cheers,
Jörg
