Dear DeepaMehta users,

Danny has recently written a plugin (dmx-weblient [1]) that
allows you to develop new clients for DeepaMehta from within DeepaMehta.
The goal is to have a live-coding environment for rapid application
development and prototyping. 

You can download and install the plugins (dmx-webclient, webactivator)
together with the latest nightly builds from the demo server [2].
