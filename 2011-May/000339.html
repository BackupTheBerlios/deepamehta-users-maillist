Semantic Web: Zwischen Formaler Ontologie und Künstlicher Intelligenz

Das World Wide Web stößt an die Grenzen seiner Brauchbarkeit und Zuverlässigkeit. 
Eine Stichwortsuche liefert als Ergebnis eine Liste von Webseiten unterschiedlicher Provenienz, die anschließend mühsam durchkämmt werden müssen. 
Ziel des Semantic Webs ist es, auf gezieltere Anfragen (z.B. mittels SPARQL) hochwertigere Datasets zurückzuliefern. 

Wie üblich bei neuen Technologien ist auch das Semantic Web von einem gewissen Hype umgeben. 
Im Vortrag werden die Realität von der Fantasie getrennt und in einem generellen Überblick die Entwicklungen präsentiert. 

Zwei Ansätze zur Umsetzung des Semantic Webs werden identifiziert und kurz umrissen.
Auf der einen Seite findet sich der Ansatz von Tim Berners-Lee und seinen Mitstreitern, die versuchen, das Semantic Web auf den Grundlagen der formalen Semantik aufzubauen. 
Auf der anderen Seite ist der KI-Ansatz, der sich auf Systemen zur Verarbeitung der natürlichen Sprache (Natural Language Processing bzw. Natural Language Understanding) stützt. 
Diese Systeme sollen das Web durchsuchen und die dadurch gewonnenen Erkenntnisse zu einer Wissensbasis zusammenfügen. 
Beide Ansätze überschneiden sich teilweise, werden aber von unterschiedlichen Grundannahmen gesteuert.
