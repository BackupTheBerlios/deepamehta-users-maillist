Dear DeepaMehta friends,

on Sep 2, 2013 Web Activator 0.4 has been released.
This version works in conjunction with DeepaMehta 4.1.1 as released yesterday.

Web Activator is a DM4 plugin for server-side HTML generation based on the Thymeleaf template engine.

New in version 0.4:
	* Access session attributes from within template.
	* Concurrent requests to the same web application.
	* Deployment of more than one web application at the same time.
	* Updated Thymeleaf 2.0.16 -> 2.0.18

Download:
http://download.deepamehta.de/deepamehta-webactivator-0.4.jar

Source Code:
https://github.com/jri/dm4-webactivator

Simple example project that demonstrates developing a web application with DM4 Web Activator:
https://github.com/jri/dm4-example-webapp

Thymeleaf template engine:
http://www.thymeleaf.org/

Cheers,
Jörg
