On September 27, 2011 DeepaMehta 4.0.4 has been released.

Main features of this release are
- file management
- GUI customization
- handling larger topic amounts
- daemon support.
For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

You must start with a fresh database.

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.4.zip

Installation instructions in the README:
https://github.com/jri/deepamehta#readme

Build from Source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de


Thanks to Christiane Müller, Daniel Paufler, Danny Gräf, Ingo Rau, Jürgen Neumann, Malte Reißig, and Silke Meyer for contributing to this release!

Cheers,
Jörg
