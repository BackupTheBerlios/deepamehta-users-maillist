Quite a long time has passed since the release of DM 4.1. We don't want leave you in uncertainty any longer and report about the upcoming 4.1.1 release here. The new 4.1.1 features are mostly related to the DM framework as they are required by the EduZEN applications.

Main features of DM 4.1.1:

- Timestamps.
Every topic and association have "created" and "last modified" timestamps. DM Applications can access timestamps and perform queries via a new Time API.
https://trac.deepamehta.de/ticket/386

- Edit Conflict Detection.
For collaborative work: When two users edit one topic at the same time no updates are lost. Instead the server rejects the conflicting update request.
https://trac.deepamehta.de/ticket/479

- Performance.
Exploiting the browser's own cache results in lesser network traffic and better perceived performance. Intrinsic HTTP features are consequently used.
https://trac.deepamehta.de/ticket/478

DM 4.1.1 is about to be released this month.
