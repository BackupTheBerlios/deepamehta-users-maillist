On October 20, 2011 DeepaMehta 4.0.5 has been released.

There are crucial Windows and Apache related bug fixes and 3 hot features:
	- Custom topicmap backgrounds
	- Functional browser back/forward buttons
	- Permalinks
For details see the release notes:
https://trac.deepamehta.de/wiki/ReleaseNotes

You must start with a fresh database.

Download:
https://github.com/downloads/jri/deepamehta/deepamehta-4.0.5.zip

Installation instructions:
https://github.com/jri/deepamehta#readme

Build from source:
https://trac.deepamehta.de/wiki/BuildFromSource

For reporting issues please register at our trac-site:
https://trac.deepamehta.de

Thanks to Christiane Müller, Danny Gräf, Ingo Rau, Jürgen Neumann, Malte Reißig, Silke Meyer, and Rolf Kutz for contributing to this release!

Cheers,
Jörg
