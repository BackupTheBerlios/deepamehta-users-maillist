Dear DeepaMehta users,

the Ubuntu/Debian package of the latest stable release is now available
here:

https://launchpad.net/~sil-ke/+archive/deepamehta4

deb http://ppa.launchpad.net/sil-ke/deepamehta4/ubuntu
YOUR_UBUNTU_VERSION_HERE main

Ubuntu users can thus execute add-apt-repository ppa:sil-ke/deepamehta4

Unfortunately, this is not the same PPA I set up last week when I
uploaded a package of the 4.0.5-snapshot. So those of you who had
already installed the snapshot will have to rename the PPA from
deepamehta to deepamehta4. Sorry for that!

Feedback is very welcome!

Cheers, Silke
