Hi JuergeN,

From my side (since Facets and Geomaps are included in the DM Standard Distribution) there are just 3 external plugins:

- DM4 Kiezatlas: A geographical content management system. A rewrite of Kiezatlas 1.6
  https://github.com/jri/dm4-kiezatlas

- DM4 Web Activator: Server-side HTML generation based on the Thymeleaf template engine.
  https://github.com/jri/dm4-webactivator

- DM4 Example Webapp: A very simple web application developed with DM4 Web Activator.
  https://github.com/jri/dm4-example-webapp

These plugins (except the 3rd) are listed on the website's Download page:
https://www.deepamehta.de/en/content/download
We could keep the plugin list (including a one-line description for each plugin) up-to-date there.
